# 🎮 Contador Gamer 

Projeto desevolvido em React Native com Expo

--- 

# Preview do APP

O aplicativo possui:
* contador de pontos;
* botão de aumentar;
* botão de diminuir;
* botão de resetar;
* interface estilo gamer.

---

# Tecnologia utilizadas

* React Native
* Expo
* JavaScript

---

# Como executar o projeto

## 1. clonar o repositório
```
git clone https://github.com/euleoh/contadorGamer.git
```

## 2. Entrar na pasta  do projeto
```
cd contadorGamer
```

## 3. Instalar dependências
```
npm install
```

## 4. Rodar o projeto
```
npm start
```
